# Smart Alerts Dashboard

A modern dashboard built with **Next.js**, **TypeScript**, and **Tailwind CSS**. This project demonstrates professional front-end architecture and real-time data filtering.

## Key Features
- **Real-time Search:** Filter alerts instantly.
- **Dynamic UI:** Smart cards with logic-based styling (Parity ID logic).
- **Shadcn/UI:** Professional and accessible components.
- **Responsive:** Mobile-first design.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Shadcn/UI
- Vercel (Deployment)

## Project Structure
- `/app`: Routing and layouts.
- `/components`: Reusable UI elements.
- `/lib`: Utilities and helpers.

## How to Run
1. `npm install`
2. `npm run dev`
